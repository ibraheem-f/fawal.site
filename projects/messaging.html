Project Overview

This group project was developed using Java socket programming. It enables users to register, log in, and chat in real time across a TCP/IP connection using a custom client-server architecture. Each message is structured and serialized for transmission.

Key Features

  • Client-server message handling with threading
  • User login, logout, and messaging
  • Message classes with message type handling
  • Terminal-based interface (or GUI optional)
