Project Overview

This card game app pits the user against a computer opponent. Users draw cards and try to win rounds. The game tracks scores and features smooth animations and a polished UI developed with Android Studio.

Key Features

  • Turn-based gameplay vs. AI bot
  • Score tracking + animations
  • Player + bot deck UI with flipping effect
  • End game stats tracking and toast notifications
