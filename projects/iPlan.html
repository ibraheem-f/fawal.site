Project Overview

This iOS app mimics iMessage’s clean interface while allowing users to plan hangouts and dates. Users can propose times, vote, and finalize a time for the group to meet — all within a sleek UI familiar to iPhone users.

🌐 View on GitHub

Key Features

  • iMessage-style interface using SwiftUI
  • Time proposal + voting + finalization
  • Local notification reminders
  • Fully responsive for iPhone screen sizes
